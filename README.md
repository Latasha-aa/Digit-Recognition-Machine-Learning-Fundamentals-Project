# 🔢 Handwritten Digit Recognition (CNN)

## 📌 Project Overview

This project implements a **Handwritten Digit Recognition system** using a **Convolutional Neural Network (CNN)** to classify digits (0–9) from image data. Handwritten digit recognition is a fundamental computer vision problem where models learn to interpret human-written numbers from images. ([Devpost - The home for hackathons][1])

The model is trained on image data and optimised to achieve accurate multi-class classification through preprocessing, training, and evaluation.

---

## 🚀 Key Features

* 🧹 **Data Preprocessing**

  * Normalized pixel values for better convergence
  * Reshaped image data for CNN input
  * Ensured clean and structured dataset

* 🧠 **Model Development**

  * Built a Convolutional Neural Network (CNN)
  * Used layers like Conv2D, MaxPooling, and Dense
  * Applied activation functions for non-linearity

* 📈 **Model Evaluation**

  * Evaluated using accuracy and loss metrics
  * Validated model on unseen test data
  * Analyzed prediction performance

* ⚙️ **Optimization**

  * Tuned hyperparameters (epochs, batch size)
  * Reduced overfitting using regularization techniques
  * Improved model generalization
